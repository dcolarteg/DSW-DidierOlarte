Propósito:

El proposito de este sitio es fomentar el aprendizaje de diseño de sitios web, enfocándonos en las temáricas de:
Etiquetas semánticas, Listas y enlaces.


Etiquetas semàticas

Las etiquetas semánticas ayudan a definir la estructura del documento y permiten que las páginas web sean mejor indexadas por los buscadores. Una etiqueta se califica como semántica si tiene que ver el significado, es decir, si nos informa sobre lo que trata su contenido.



Listas y Enlaces

Las listas en HTML nos permite crear conjuntos de elementos en forma de lista dentro de una página, todos los cuales irán precedidos, generalmente, por un guión o número. Los tipos de listas en HTML son los siguientes: Listas ordenadas. Listas desordenadas.
